============
gene_exp_psi
============

This repository contains the scripts used to create the results for *Pervasive Coupling of Gene Expression and Alternative Splicing with Two Classes of Exon Skipping Events*.
The entire analysis can be run as a Snakemake pipeline, or the scripts can be run independently.
